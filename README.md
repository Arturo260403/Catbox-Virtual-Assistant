# CATBOX — Asistente virtual / Virtual Assistant

**ES:** CATBOX es un asistente virtual en formato web diseñado para **resolver consultas sobre datos internos**, **realizar cálculos** y **recuperar información** de forma rápida, clara e intuitiva.  
**EN:** CATBOX is a web-based virtual assistant built to **answer questions about internal data**, **perform calculations**, and **retrieve information** in a fast and user-friendly way.

---

## Visión general / Overview

### El desafío / The challenge
**ES:** Crear un chatbot capaz de responder preguntas sobre datos internos, realizar cálculos y consultar información.  
**EN:** Build a chatbot capable of answering questions about internal data, performing calculations, and retrieving information.

### Objetivo / Goal
**ES:** Desarrollar una aplicación web moderna con una interfaz intuitiva y un asistente inteligente.  
**EN:** Develop a modern web application with an intuitive interface and an intelligent assistant.

---

## Funcionalidades principales / Key features
- **ES:** Chat interactivo para consultas y soporte.  
  **EN:** Interactive chat for queries and support.
- **ES:** Cálculos y respuestas automatizadas según la información disponible.  
  **EN:** Automated calculations and responses based on available information.
- **ES:** Interfaz web moderna y enfocada en usabilidad.  
  **EN:** Modern, usability-focused web interface.
